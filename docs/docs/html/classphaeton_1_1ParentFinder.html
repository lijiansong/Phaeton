Detailed Description

ParentFinder - This pass can help to lift out expressions below contractions. It may be helpful for testcase 'inverse Helmholtz'; FIXME: whether this pass is generally a useful trnasformation for all testcases remains to be determined.

Note: there has also been a fix in the code that fuses two subsequent assignments, i.e. definitions of local variables for the second assignment are now emitted(code generated) before the first assignment. Note that this may not work in general, i.e. if more than two assignments are fused.
